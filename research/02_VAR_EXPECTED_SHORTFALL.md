# VaR and Expected Shortfall

The project implements several base tail-risk estimators:

- historical VaR;
- historical Expected Shortfall;
- parametric Gaussian VaR/ES;
- Cornish-Fisher VaR adjustment;
- Student-t VaR/ES approximation;
- EVT/POT proxy for extreme losses.

Expected Shortfall is emphasized because it measures the average loss conditional on being in the tail, whereas VaR only reports a threshold.
